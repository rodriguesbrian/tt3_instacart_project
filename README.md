# tt3_Instacart_project

Este projeto analisa dados da **Instacart**, plataforma de delivery de supermercado, para explorar os hábitos de compra dos clientes e gerar insights sobre padrões de pedidos.

---

## 🧪 Objetivo

Investigar o comportamento de compra dos clientes da Instacart, incluindo horários de pedidos, frequência de produtos comprados e padrões de pedidos repetidos.

O objetivo do projeto é **limpar, processar e analisar os dados** para responder perguntas específicas sobre hábitos de compra.

---

## 🔍 Descrição do Projeto

O projeto está dividido em três etapas principais:

1. **Etapa 1 – Exploração de Dados:**  
   Visualização inicial e inspeção dos arquivos CSV (`orders`, `products`, `order_products`, `aisles`, `departments`) para entender estrutura e qualidade dos dados.

2. **Etapa 2 – Pré-processamento:**  
   Limpeza e preparação dos dados:  
   - Correção de tipos de dados  
   - Tratamento de valores ausentes  
   - Remoção de duplicados  
   - Documentação das decisões de pré-processamento no notebook

3. **Etapa 3 – Análise Exploratória:**  
   Análise e visualização de dados para responder perguntas como:  
   - Distribuição de pedidos por hora e dia da semana  
   - Intervalo entre pedidos  
   - Produtos mais comprados e mais repetidos  
   - Produtos adicionados primeiro no carrinho
